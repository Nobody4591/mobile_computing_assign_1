Git and GitHub for beginners

Learning Git is just as easy as doing some simple commads, knowing what you've to do. we'll start by first creating a folder, so that we can convert it as Git repository. Anything you wanna keep track off, just put it in that folder, and let the know whether you wanna include that perticular thing or not.

STEP 1

let's create the folder, commad for creating the folder is:
mkdir git_practice

STEP 2

Inititate the Git repository in the folder, the command for Inititating the repository is as simple as
git init

STEP 3

we've created the git repository successfully. Now, we just have to add things in the folder,
which we want to be tracked.
let's create a file, name user_manual.html
comand for creating the file is, touch user_manual.html

STEP 3

Now, we've to understand some working of Git.
we've created a file in the git repository, in same way, we can add files as much as we want. there are three stages
for a git repository,

  • Add the file in git repository ( simple words, in Git negivation).
  • Stage the current file, so we can have idea what go store in the next commit. Although,this step is optional
  • Commit(store the file permanatly) the file in the Git negivation
while working on the files, we can check the status of files, we also have command for that.
let give it a try
we've added the file in folder but neither we, add the file git repository nor Commit
  1. Adding File in Git repository

    adding the file in git repository is easy with the command,
    "git add 'filename'" or "git add ." in our case, it like git add user_manual.html

  2. Stage File in git repository

    adding file in git is necessary but it doesn't meet the requirement. it just now track which file
    is changes and where.
    for the file changes to be permanat, we have to either stage the situation of File. Think it as
    you wanna some more rough work today, but don't want that work to be stored or parmanent
    So, stage commands will help you achieve this.
    git stage

  3. Commit the File

    last think we have to do is to make all the change patrmaent, git coomit command,
    can help in this.
    git commit .
